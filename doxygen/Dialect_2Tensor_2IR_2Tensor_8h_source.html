61 /// Determines whether tensor::CastOp casts to a more dynamic version of the
62 /// source tensor. This is useful to fold a tensor.cast into a consuming op and
63 /// implement canonicalization patterns for ops in different dialects that may
64 /// consume the results of tensor.cast operations. Such foldable tensor.cast
65 /// operations are typically inserted as `extract_slice` ops and are
66 /// canonicalized, to preserve the type compatibility of their uses.
67 ///
68 /// Returns true when all conditions are met:
69 /// 1. source and result are ranked tensors with same element type and rank.
70 /// 2. the tensor type has more static information than the result
71 ///
72 /// Example:
73 /// ```mlir
74 /// %1 = tensor.cast %0 : tensor<8x16xf32> to tensor<?x?xf32>
75 /// %2 = consumer %1 ... : tensor<?x?xf32> ...
76 /// ```
77 ///
78 /// folds into:
79 ///
80 /// ```mlir
81 /// %2 = consumer %0 ... : tensor<8x16xf32> ...
82 /// ```
83 bool canFoldIntoConsumerOp(CastOp castOp);
