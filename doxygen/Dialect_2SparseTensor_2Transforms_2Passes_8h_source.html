24 /// Defines a parallelization strategy. Any independent loop is a candidate
25 /// for parallelization. The loop is made parallel if (1) allowed by the
26 /// strategy (e.g., AnyStorageOuterLoop considers either a dense or sparse
27 /// outermost loop only), and (2) the generated code is an actual for-loop
28 /// (and not a co-iterating while-loop).

30  kNone,
31  kDenseOuterLoop,
32  kAnyStorageOuterLoop,
33  kDenseAnyLoop,
34  kAnyStorageAnyLoop

36 };

38 /// Defines a vectorization strategy. Any inner loop is a candidate (full SIMD
39 /// for parallel loops and horizontal SIMD for reduction loops). A loop is
40 /// actually vectorized if (1) allowed by the strategy, and (2) the emitted
41 /// code is an actual for-loop (and not a co-iterating while-loop).
42 enum class SparseVectorizationStrategy {
43  kNone,
