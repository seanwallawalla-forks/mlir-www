Detailed Description

ToMemrefOp casts a tensor into a memref.

The resulting memref is the memory location of the incoming tensor once it will be bufferized. In the anlysis, the incoming tensor is assumed to bufferize to a memory read and to an inplace memory write, since it is unknown what will happen to the resulting memref.
